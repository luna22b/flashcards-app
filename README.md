## ✏️ Lume - Flashcards App

Lume is a full-stack flashcard application that helps users create, organize, and study flashcards through multiple learning modes.


## 🛠️ Technologies Used
- React
- TypeScript
- Prisma
- TailwindCSS
- PostgreSQL
- Node.js
- Express

## ✨ Features
- Secure user authentication
- Create, edit, and delete flashcards sets
- Add, edit, and remove individual flashcards
- Multiple-choice quizzes with shuffled questions
- Persistent storage with PostgreSQL

## 💡 Why I Built Lume
I wanted a application that I will personally use. I wanted to pursue a much more
simplistic approach to the flashcards app to make it much more practical and distraction free
to study. I saw that some flashcard applications may have subscriptions to unlock certain
features, so I decided to build my own app instead of paying. It will incorporate all the
features that I use, while also thinking about the user experience. 

## 🚦 Future Improvements
- Spaced repetition algorithm for flashcards mode
- Import and export flashcard sets
- A learn mode to ensure the best practice
- Dark mode

## 🚀 Getting Started

1. Clone the repository.

```bash
git clone https://github.com/your-username/lume.git
```

2. Install the dependencies.

```bash
npm install
```

or

```bash
yarn
```

3. Start the development server.

```bash
npm run dev
```

4. Open your browser and navigate to (or the address in your console):

```
http://localhost:3000
```
